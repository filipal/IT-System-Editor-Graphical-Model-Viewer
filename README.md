Web application for visualizing and editing modeled IT systems based on JSON files.

The app allows uploading multiple JSON models and displays them as an interactive network graph where computers, software, users, and services are visually connected and editable.

Built as a graphical editor for exploring and modifying complex IT system structures.

---

## What this app demonstrates
- Upload and management of multiple JSON model files  
- Automatic transformation of JSON data into a network graph  
- Interactive graphical editing of IT system components  
- Dynamic updates of graph structure based on edits  
- Custom graph views and filtering logic  

---

## Tech stack
- React  
- TypeScript  
- Reagraph (graph visualization)  
- Custom graph-processing logic
